캐시워크 돈버는퀴즈 정답 확인 전 알아둘 점

편집팀 · 상시 업데이트

캐시워크 돈버는퀴즈는 같은 날에도 여러 브랜드와 문항이 함께 노출될 수 있습니다. 정답을 확인할 때는 날짜만 보지 말고 문제 문장 속 키워드, 브랜드명, 숫자 단서를 함께 비교하는 것이 좋습니다.

정답이 다르게 보일 수 있는 이유

퀴즈는 이벤트 진행 상황에 따라 문항이 교체되거나 정답 표기가 달라질 수 있습니다. 또한 사용자에게 노출되는 문항이 서로 다를 수 있으므로, 이 사이트의 글은 앱 화면과 대조해 확인하는 보조 자료로 활용하세요.

날짜별 글 작성 팁

새 글을 만들 때는 제목에 날짜를 넣고, 본문에는 브랜드명, 문제 키워드, 정답, 업데이트 시간을 함께 남기는 편이 검색과 관리에 유리합니다. 변경된 정답이 있으면 기존 정답을 지우기보다 업데이트 메모를 추가하면 방문자가 혼동을 줄일 수 있습니다.

ChatGPT 무료버전으로 정리하는 방법

ChatGPT 무료버전은 이 사이트에서 자동 호출할 수 있는 API가 아닙니다. 대신 캐시워크 앱에서 직접 확인한 문제 문장과 정답 후보를 ChatGPT에 붙여넣고, 아래 형식으로 JSON을 만들어 달라고 요청한 뒤 data/answers.json에 반영하면 날짜별 페이지에 자동 표시됩니다.

아래 캐시워크 돈버는퀴즈 정보를 JSON으로 정리해줘.
추측하지 말고 내가 준 정보만 사용해.

날짜: 2026-06-03
업데이트 시간: 09:00
브랜드/문항:
1. 브랜드명:
   문제 문장:
   정답:
   메모:

원하는 JSON 형식:
{
  "date": "YYYY-MM-DD",
  "updatedAt": "HH:mm",
  "title": "YYYY년 M월 D일 캐시워크 돈버는퀴즈 정답",
  "url": "answers/YYYY-MM-DD.html",
  "summary": "요약 문장",
  "items": [
    {
      "brand": "브랜드명",
      "question": "문제 문장",
      "answer": "정답",
      "note": "확인 메모"
    }
  ]
}

GitHub Actions 자동화

완전 자동화는 GitHub Actions가 정답 후보 URL을 읽고, OpenAI API로 오늘 날짜의 정답 JSON을 정리한 뒤 저장소에 자동 커밋하는 방식으로 동작합니다. 설정은 저장소의 Actions Secrets와 Variables에서 관리하며, API 키는 공개 코드에 넣지 않습니다.
